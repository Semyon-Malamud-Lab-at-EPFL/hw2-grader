# Assignment 2 Grader (Private)

## Setup
1. Create private repo `hw2-grader` under your org.
2. Push all files from this directory.
3. Create a PAT (fine-grained, read-only on `hw2-grader`).
4. Add it as org secret `GRADER_PAT`.
5. Place `data.csv` in the student template's `data/` directory.
6. Create a GitHub Classroom assignment from the student template.

## ddof policy
Task 7b (vol, sharpe) accepts **both** ddof=0 and ddof=1.
